# ServiceHub

Aplicação monolítica **Laravel + Vue 3** com **MySQL**, containerizada via **Docker**.  

Este projeto é um ambiente de desenvolvimento profissional, pronto para rodar localmente sem configurar PHP, Node ou MySQL manualmente.

---

## 🛠 Tecnologias

- **PHP 8.2 + Laravel (latest stable)**  
- **Vue 3 + Vite**  
- **MySQL 8**  
- **Nginx**  
- **Docker + Docker Compose**
